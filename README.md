https://frycards.herokuapp.com/about

The service for learning foreign words by cards. After every guessed card, the time of showing this word again will encrease..
You can add and manage decks, cards and other.

Rails 4.1.6

DB - Postgresql

Authorization - Sorcery

Testing - (Rspec, Capybara, Factory Girl)

Images upload - Amason S3
